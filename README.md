# Neon Snake

A polished, arcade-style Snake game built with vanilla HTML, CSS, and JavaScript.

## Features

- Classic Snake mechanics (grid movement, growth, collision game-over)
- Keyboard controls (Arrow keys + WASD)
- Touch-friendly on-screen controls
- Pause/Resume and Restart controls
- Persistent best score in browser storage
- Pure, deterministic core logic with tests

## Controls

- Move: `Arrow Keys` or `W A S D`
- Pause/Resume: `Space` or `P`
- Restart: `R`
